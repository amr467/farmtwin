🌳 FarmTwin Tree Counter — Setup

  1. Make sure you have Python installed on your computer
  2. Open a terminal / Command Prompt and run:
    pip install flask flask-cors pillow numpy requests opencv-python scipy
  3. Download tree-counter.py and double-click start.bat
    — or run: python tree-counter.py
  4. Come back here — the page will connect automatically
ℹ The server runs on your computer at localhost:5050.
Once started, this overlay disappears and you can begin drawing areas.
